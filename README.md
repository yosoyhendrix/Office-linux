# Cómo usar este script

### 1. Copia el código de arriba y pégalo en un archivo de texto llamado instalar_office.sh
### 2. Abre tu terminal.
### 3. Ve a la carpeta donde guardaste el archivo.
###4. Dale permisos de ejecución con el comando:
``` chmod +x instalar_office.sh```
### 5. Ejecútalo:
``` ./instalar_office.sh```
(Te pedirá tu contraseña de usuario porque necesita usar sudo para instalar los programas).
